# Flask_Blog

Pavly Salah's submission for the Digizilla task using Flask.

- Required libraries:
    - Flask
    - WTForms
    - Flask_login
    - SQLAlchemy
    - Flask_Bcrypt
    - Flask_Mail

- Features:
    - Create user
    - Update user info
    - User authentication
    - Reset password
    - Upload profile picture (and save it locally)
